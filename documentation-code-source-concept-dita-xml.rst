Concept DITA XML
================

Les sections **concept** permettent au **rédacteur technique** de présenter une
introduction des sections *task* ou *reference*.

Exemple de structure d'un *concept* :

.. aafig::

                +-title
                |
      concept--shortdesc
                |
                +-conbody
